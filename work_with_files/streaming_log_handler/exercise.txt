Цель: Научиться обрабатывать файлы построчно, не загружая весь файл в память. Идеально для логов (NDJSON).

1. Создай вручную файл "events.ndjson" с таким содержимым:
    
    """python
    {"event": "login", "user_id": 1}
    {"event": "click", "user_id": 2, "target": "button_buy"}
    {"event": "login", "user_id": 3}
    {"event": "logout", "user_id": 1}
    """
    
2. Напиши функцию "count_logins(log_path: Path) -> int", которая считает, сколько раз в файле произошло событие ""login"".
3. Функция должна:
    - Открыть файл "log_path" в режиме ""r"" с "encoding="utf-8"".
    - Итерироваться по файлу "for line in f:".
    - Внутри цикла:
        - Парсить каждую "line" с помощью "json.loads()".
        - Проверять, равно ли "event.get("event")" значению ""login"".
        - Увеличивать счетчик.
    - Вернуть итоговый счетчик.
    - (Бонус): Оберни "json.loads(line)" в "try...except json.JSONDecodeError", чтобы пропустить "битые" строки лога, не прерывая всю обработку.
